📘 Materi Pertemuan 14 - Menentukan Titik Potong Garis dengan Sumbu X dan Y

📘 Pemahaman Konsep

Persamaan garis lurus memiliki bentuk umum:

y = mx + c
  • m adalah gradien (kemiringan garis)
  • c adalah intersep (titik potong dengan sumbu y)

Untuk menentukan titik potong garis dengan sumbu x dan y:

  • Titik potong dengan sumbu y terjadi saat x = 0
  • Titik potong dengan sumbu x terjadi saat y = 0

Langkah-langkah:

  1. Substitusikan x = 0 ke dalam persamaan untuk mendapatkan titik potong dengan sumbu y
  2. Substitusikan y = 0 ke dalam persamaan untuk mendapatkan titik potong dengan sumbu x

🧠 Contoh Soal Kontekstual dan Penyelesaiannya

Contoh Soal:

Sebuah perusahaan pengiriman mengenakan biaya tetap Rp10.000 dan tambahan Rp2.000 per kilometer. Persamaan biaya pengiriman adalah y = 2000x + 10000, dengan x adalah jarak (km) dan y adalah biaya (Rp).

Tentukan titik potong garis dengan sumbu x dan y, dan interpretasikan maknanya.

Penyelesaian:

  • Titik potong dengan sumbu y: x = 0 → y = 10000 → Titik: (0, 10000)
  • Titik potong dengan sumbu x: y = 0 → 0 = 2000x + 10000 → x = -5 → Titik: (-5, 0)

Interpretasi:

  • Titik (0, 10000) menunjukkan biaya tetap minimum yaitu Rp10.000
  • Titik (-5, 0) tidak bermakna secara kontekstual karena jarak tidak bisa negatif, namun secara grafik menunjukkan perpotongan dengan sumbu x

📝 Latihan Soal

  1. Diketahui persamaan garis y = -3x + 12. Tentukan titik potong dengan sumbu x dan y.
  2. Persamaan garis y = 5x - 20 menunjukkan hubungan antara jumlah barang dan harga. Tentukan titik potong dengan sumbu x dan y, lalu interpretasikan maknanya.

🔍 Refleksi Pembelajaran

  1. Apa yang dimaksud dengan titik potong garis dengan sumbu x dan y?
  2. Bagaimana cara menentukan titik potong tersebut dari persamaan garis?
  3. Dalam konteks kehidupan nyata, apa makna dari titik potong dengan sumbu y?
